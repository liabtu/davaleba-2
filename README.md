# Student Form — Jetpack Compose

სტუდენტის საინფორმაციო ფორმა შექმნილი Jetpack Compose-ით. პროექტში შეცვლილია სტანდარტული დიზაინი (Unique UI) და რეალიზებულია სრული ვალიდაცია.

## 📱 აპლიკაციის დემონსტრაცია

![Demo](ჩასვი_შენი_gif_ბმული)

## 🛠️ ფუნქციონალი:
* **Custom UI:** მორგებული ფერები, მომრგვალებული ველები და იერსახე.
* **DatePicker:** თარიღის არჩევა ფორმატით `DD/MM/YYYY`.
* **ვალიდაცია:** "Submit" ღილაკზე დაჭერისას მოწმდება ყველა ტექსტური ველი, RadioButton და Switch. შეცდომისას გამოდის Toast: "შეავსეთ ყველა ველი!", წარმატებისას: "მონაცემები გაიგზავნა!".
